# Lo Imaginas | Portafolio en Vue

Migración de portafolio estático (HTML/CSS/JS) a **Vue 3 + Vite** con arquitectura por componentes y vistas.

## Estructura principal

- `public/` recursos públicos (`favicon.ico`, `patita.webp`)
- `src/assets/img/` imágenes del sitio y proyectos
- `src/components/` componentes reutilizables
- `src/views/` vistas por ruta
- `src/router/index.js` configuración de rutas
- `src/styles/main.css` estilos globales
- `src/App.vue` layout general
- `src/main.js` bootstrap de Vue

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Notas

- La app quedó lista para seguir creciendo por módulos.
- Las imágenes de proyectos (`proyecto-*.png`) se dejaron como placeholder para que luego puedas reemplazarlas por capturas reales.
